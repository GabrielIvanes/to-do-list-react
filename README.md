# to-do-list-react

To do list made with vite, TypeScript and React. You can create, delete, filter and edit task. Tasks are saved in the local storage. 

Getting started : 
```sh
$ npm create vite@latest
```

```sh
$ npm install @fortawesome/fontawesome-svg-core
$ npm install @fortawesome/free-solid-svg-icons
$ npm install @fortawesome/react-fontawesome
```
